使用Hexo3+GitHub搭建个人博客

  • hexo

一直以来想写点东西记录自己的点点滴滴，却发现没有比较适合的，就萌发了自己要搭建一个属于自己的博客的想法，然后，一不小心就遇见了Hexo。

什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

安装 Hexo 只需几分钟时间，若您在安装过程中遇到问题或无法找到解决方式，Google一下。

安装前提

安装 Hexo 相当简单。然而在安装前，须检查电脑中是否已安装下列应用程序：

  • Node.js

  • Git

如果您已经成功的安装，那么恭喜您！接下来只需要使用npm即可完成Hexo的安装。

Layout Destination
post(Default) source/_posts

post用来放自己写的文章。
page可以用来放诸如关于我，友情链接，404之类的自定义页面。
比如关于我页面，使用hexo new page "about"命令会在生成about目录，里面有个index.md，直接编辑就可以了，然后在blog的_config.yml中将其配置显示出来。

404页面
GitHub Pages 自定义404页面非常容易，直接在根目录下创建自己的404.html就可以。但是自定义404页面仅对绑定顶级域名的项目才起作用，GitHub默认分配的二级域名是不起作用的，使用hexo server在本机调试也是不起作用的。

其实，404页面可以做更多有意义的事，来做个404公益项目吧。

  • 腾讯公益404
  • 404公益_益云(公益互联网)社会创新中心
  • 失蹤兒童少年資料管理中心404
